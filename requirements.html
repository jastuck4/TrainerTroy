Your site includes an XML Sitemap		 Contrast: Active pages are made to have a different background color and height from inactive pages. Images that are hovered over within the gallery are made to be enlarged so that it stands out from those that are not being viewed.

Proximity: Client testimonials are made to connect to the client’s image to show relationship. The additional links are all grouped together in a list within the footer. The validation information for the pages are also grouped together within the footer. The content is divided by the use of headers so that the user knows what each section is.

Alignment: I used the end of the search bar and the beginning of the navigation bar as markers to note where to place certain elements.

Repetition: I repeated all style elements on all pages. All main headers are greatly enlarged to the same size and all subject headers have a different background color from other elements on the site.

The site navigation links are clearly named for what that particular page is about. Whenever a page is active, it distinctly stands out from the rest so that the user knows where they are.

The font color is white on a black background for the main content and white on a red background for the navigation and footer. The header image’s text is also easy to read with the font being black and dark grey on a white background. The font style is Arial, which is easy to make out. I also adjusted the line height so that the spacing between paragraph lines doesn’t tire the eyes as much. Even though I have the main content on a textured background, it does not interfere with readability of content because of the subtlety.

The text and graphics are placed according to relationship. The gallery page contains graphics that relate to fitness transformation and motivation, the about me page contains graphics highlighting the personal trainer, and the contact page has a graphic motivating users to start training. I also included a header image that stands out but is not visually overwhelming. All images are proportional, clear, and professional looking. All pages are designated to a specific topic. Graphics that are included on some of the pages are only included as they relate to the content of that page. I also made active pages the same background as the main content so that it stands out from the inactive pages, which remains with a red background.
